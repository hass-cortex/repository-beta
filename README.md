# Hass Cortex Apps (Beta)

Beta-channel catalog of [hass-cortex](https://github.com/hass-cortex) Home Assistant apps.

This catalog is a **superset** of the stable channel: it receives every stable release plus pre-releases (`-beta.N`, `-rc.N`, etc.) for early testing.

For production use, install the stable catalog instead: <https://github.com/hass-cortex/repository>.

## Install in Home Assistant

1. Open **Settings → Apps → App Store** in Home Assistant.
2. Click the three-dot menu → **Repositories**.
3. Add this repository URL:
   ```
   https://github.com/hass-cortex/repository-beta
   ```
4. Refresh the App Store. Apps will appear.
